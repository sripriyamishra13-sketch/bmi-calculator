# Advanced BMI Calculator

A graphical Body Mass Index (BMI) calculator built using Python and Tkinter.  
This application allows users to calculate BMI, store historical records, and visualize trends through graphs.

## Tech Stack

- Python 3.11
- Tkinter (GUI)
- Matplotlib (Data Visualization)
- CSV (Data Storage)
- Git & GitHub (Version Control)

---

## Features

- Interactive GUI interface
- BMI calculation with health categorization
- User input validation
- Multi-user data storage
- Historical BMI tracking
- Trend visualization using graphs
- Error handling for invalid inputs

## Run Locally

### Create virtual environment
python -m venv venv  
venv\Scripts\activate

### Install dependencies
pip install -r requirements.txt

### Run
python app.py
